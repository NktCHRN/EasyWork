What is this?

This is a Gantt chart of your project. With it, you can easily control how effectively your team members are working on tasks in order to plan your further work and schedule new tasks smarter.

To generate a task, you project's lifetime should be at least one day.

At the top of the chart you can see a timeline. If you project has been existing for more than one 365 days, it will start from the day that was 365 days ago. In another case, it will start from the day when the project was created.
The end of the line is the current date.

Most likely, you do not see the full timeline on your screen, so, you can scroll the chart left or right. If you have too many tasks on the chart, you will also be able to scroll it up and down.

You can also see some orange rectangles on your chart, like this one:

Sample task

They are project tasks. Each rectangle starts at the date when the task was created and ends at the 'End date', which you can see and edit on the Task dialog. If the end date is not defined, the rectangle will be continued up to the end of the timeline. If the name of the task is too big, it will be cut and end with three dots (...), but you can view the full name by hovering on it. Task rectanges are not clickable.

This chart is not updated dynamically when someone creates, changes or deletes a task. So, you need to reload this page in order to see the changes
